Natural Language Prompt
"Can you review this PR and let me know if it's safe to merge?"
Anthropic Prompt Framework Template
Role & Objective: Review the diff and decide merge readiness. Capabilities & Tools: Static analysis only; no network calls. Constraints & Safety: No secrets in code; flag insecure patterns. Process: List risks → check tests → verify coverage → final decision. Input Format: Unified diff text + test report summary. Output Format: JSON { decision: "merge|block", reasons: [strings], risks: [strings] }. Success Criteria: Decision justified by risks and tests; no missing blockers.

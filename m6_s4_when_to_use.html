When to Use ML‑Friendly vs Natural Language
Use ML‑Friendly When…
  • Outputs must be reproducible and testable
  • Agents/tools consume structured outputs
  • Safety/compliance constraints apply
  • Metrics and SLAs matter
  • Golden/eval tests are required
Use Natural Language When…
  • Brainstorming and ideation
  • Exploratory research
  • Open‑ended creative writing
  • Conversational guidance
  • Low‑stakes, human‑in‑the‑loop tasks
Rule of thumb: default to Anthropic Framework → relax to natural language only when exploration outweighs precision.

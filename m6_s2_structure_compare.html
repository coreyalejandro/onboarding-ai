Structure: Natural Language vs Anthropic Prompt Framework
Natural Language Prompt
  • Conversational prose
  • Implicit constraints
  • Flexible output shape
  • Open‑ended reasoning
"Summarize the attached document and tell me the most important parts. If anything is unclear, ask me a question."
Anthropic Prompt Framework (Default)
Role & ObjectiveWhat you are; the task to achieve
Capabilities & ToolsWhat you can use
Constraints & SafetyScope limits and refusal rules
ProcessHigh‑level plan/steps
Input FormatHow input arrives
Output FormatExact schema (e.g., JSON)
ExamplesFew‑shot inputs/outputs
Success CriteriaDeterministic accept conditions
Role & Objective: Summarize the document into ≤ 120 words. Capabilities & Tools: None required. Constraints & Safety: Use only provided text; avoid opinions. Process: Extract key metrics → group themes → order by impact. Input Format: Raw text string. Output Format: JSON array of 4–6 bullet strings. Examples: ["…"] Success Criteria: Facts preserved; metrics included; no hallucinations.
